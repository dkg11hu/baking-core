A logic-blind, recursive graph-traversal engine designed for professional bakery production. 

## üèó The Philosophy: Logic-Blind Architecture
This engine is built on the principle that **the code should know nothing about the data's meaning.** - All fieldnames, rounding thresholds, and hierarchical relationships are defined in the **JSON Database**.
- The JS logic acts purely as a "slave" to the `definitions.json` schema.
- This allows for infinite recipe nesting (e.g., Yudane inside Sourdough inside a Final Loaf) without changing a single line of code.

## üõ† Project Structure
- `/data/Store/`: The "Database" containing the Registry and individual Product Entities.
- `/src/Graph/Core/`: The `Resolver` that handles the recursive "Series of Queries."
- `/src/Graph/Logic/`: The `Calculator` that manages high-precision scaling and physics rounding.
- `app.js`: The UI Orchestrator.
